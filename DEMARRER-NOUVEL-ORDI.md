# 🖥️ Utiliser mon app sur un nouvel ordinateur

Mémo simple à garder sous la main. Il y a **2 usages** — choisis selon ton besoin.

---

## 🅰️ Je veux juste PROSPECTER (utiliser l'app pour envoyer des courriels)

Le plus simple. Pas besoin de GitHub.

1. Copie le fichier **`Prospection-Garages-Portable.zip`** (sur ton Bureau) vers le nouvel
   ordinateur — par clé USB, ou via ton OneDrive.
2. Clic droit sur le ZIP → **« Extraire tout »**.
3. Ouvre le dossier **Prospection-Garages** → double-clic sur **`Lancer.bat`**.
4. L'app s'ouvre. Tes contacts se synchronisent tout seuls via OneDrive
   (le nouvel ordinateur doit être connecté au **même compte OneDrive**).

---

## 🅱️ Je veux AMÉLIORER l'app avec Claude (ajouter/changer des choses)

Là, on passe par **GitHub** (le casier dans le nuage).

Mon casier : **github.com/Joelbifco/prospection-garages**

1. Sur le nouvel ordinateur, ouvre **Claude** (connecté à ton compte).
2. Dis-lui simplement :

   > **« Récupère mon app depuis GitHub : github.com/Joelbifco/prospection-garages »**

3. Claude la télécharge. Vous travaillez dessus ensemble.
4. Quand c'est bon, dis :

   > **« Sauvegarde sur GitHub »**

   → Les changements sont envoyés dans le casier, disponibles sur tous tes ordinateurs.

### ⚠️ Avant de commencer à travailler sur un ordinateur
Dis d'abord à Claude :

> **« Récupère la dernière version depuis GitHub »**

Ça évite de repartir d'une vieille version si tu as modifié l'app ailleurs.

---

## 🧭 Pour t'y retrouver (les 2 nuages)

| Ce que c'est | Où ça vit | Sert à quoi |
|---|---|---|
| Le **code** de l'app | **GitHub** | L'améliorer avec Claude, de partout |
| Tes **contacts / réglages** | **OneDrive** | Synchro automatique quand tu utilises l'app |

---

## 🔑 Bon à savoir

- Ton **mot de passe Gmail n'est PAS sur GitHub** (il reste sur tes ordinateurs, via OneDrive).
- Sur GitHub, tout est **privé** : toi seul y as accès.
- Historique complet : si on casse quelque chose, Claude peut **revenir en arrière**.
- Règle d'or : **une chose à la fois** — récupère la dernière version, travaille, puis sauvegarde,
  avant de passer à un autre ordinateur.
